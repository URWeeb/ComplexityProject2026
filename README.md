# ComplexityProject2026

Алгоритмический проект по курсу Сложность Вычислений на тему "Выполнимость 3-КНФ"

Описание структуры проекта:

- Теоретическая часть проекта будет изложена в директории TheoryData.

- Программная реализация будет разбита на несколько папок(src для *.cpp файлов, include для .hpp файлов), связанных с помощью CMake для сборки на различных ОС(целевой являются дистрибутивы Linux или Unix-подобные BSD-системы и MacOS). Для проверки корректности написанного будет использоваться Unit-тестирование(библиотека GoogleTest) и CI/CD.
